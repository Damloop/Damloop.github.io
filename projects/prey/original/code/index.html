PREY – Technical Breakdown

This page showcases the core gameplay systems implemented in PREY, including movement, AI, damage, and level flow.

Player Movement System

Handles player input, rotation, movement, animations, and jump physics.

// PlayerMove.cs (excerpt)
transform.Rotate(0, x * rotationSpeed * Time.deltaTime, 0);
transform.Translate(0, 0, y * runSpeed * Time.deltaTime);

isGrounded = Physics.CheckSphere(groundCheck.position, groundDistance, groundMask);

rb.AddForce(Vector3.up * jumpHeight, ForceMode.Impulse);

Enemy AI System

Controls patrol, chase, attack behavior and animation transitions.

// EnemyMove.cs (excerpt)
if (distanceToPlayer <= attackRange)
{
    AttackPlayer();
}

Vector3 direction = (player.position - transform.position).normalized;
transform.Translate(direction * moveSpeed * Time.deltaTime, Space.World);

Damage System

Applies damage to any object with a Health component.

// Damage.cs (excerpt)
Health targetHealth = other.GetComponent();
if (targetHealth != null)
{
    targetHealth.TakeDamage(damageAmount);
}

Handles scene transitions, game over, and win conditions.

// LevelManager.cs (excerpt)
SceneManager.LoadScene(SceneManager.GetActiveScene().buildIndex);
